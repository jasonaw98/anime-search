# Anime Search App

![image](https://github.com/user-attachments/assets/d4d65b11-00d6-452d-b6b2-69ff6106bcdb)


## Features

- Search for an anime
- View the details of an anime
- Instant search with debounce
- Responsive design
- Color ranking based on the score

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Vite
- Shadcn UI
- React Router DOM
- Magic UI
- Lucide React Icons


## Run the App locally


```js
npm i 
npm run dev
```
